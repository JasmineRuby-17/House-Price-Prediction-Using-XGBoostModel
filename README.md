
# 🏡 House Price Prediction using Machine Learning (XGBoost)

## 📌 Project Overview

This project is a Machine Learning regression system that predicts house prices based on key real estate features.

## 🎯 Objective

To build a regression model that accurately predicts house prices using features such as:

* Average Area Income
* House Age
* Number of Rooms
* Number of Bedrooms
* Area Population

## ⚙️ Technologies Used

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* XGBoost Regressor
* Matplotlib
* Joblib

## 📊 Dataset

The model uses the USA Housing Dataset, which contains structured real estate data used for regression tasks.
Kaggle dataset Link : https://www.kaggle.com/datasets/praveenobulreddy/usa-housing-dataset

## 🧠 Machine Learning Workflow

1. Load and clean dataset
2. Select important features
3. Split data into training and testing sets
4. Train XGBoost regression model
5. Evaluate model performance
6. Save trained model using Joblib
7. Take user input via CLI
8. Predict house price in real-time
9. Visualize Actual vs Predicted values

## 📈 Model Performance

* R² Score: ~0.90
* MAE: Low error rate
* RMSE: Stable predictions

## 💻 Features

✔ End-to-end ML pipeline
✔ CLI-based prediction system
✔ Real-time user input prediction
✔ Model saving & loading (Joblib)
✔ Data visualization (Actual vs Predicted)
✔ Clean and modular Python code
